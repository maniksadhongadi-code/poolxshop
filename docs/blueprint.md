# **App Name**: Customer Hub

## Core Features:

- Add Customer: Add new customer to the pending customer list, including their email and phone number.
- Customer Lists: Display active and pending customers in separate lists.
- Switch Customer Status: Move customers between 'pending' and 'active' status with a 'Switch' button (requires four clicks for confirmation).
- Delete Customer: Delete pending customers from the system.

## Style Guidelines:

- Primary color: Soft blue (#89B9D1) to convey trust and reliability.
- Background color: Very light blue (#F0F4F7), close to white, for a clean and professional look.
- Accent color: Muted green (#90EE90) for success indicators and actionable elements.
- Body and headline font: 'PT Sans', sans-serif, for clear readability.
- Two-column layout: Displaying active and pending customers side-by-side.
- Simple, outline-style icons for actions like 'add', 'switch', and 'delete'.
- Subtle transition animations when switching customer status or deleting entries.
